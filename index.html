    实用工具¶

    • 二进制转换：https://tool.lu/hexconvert/

    Screen2Gif¶

    无需安装，可直接运行使用。可以吸附窗口，不用再小心翼翼地拖拉窗口了，并且提供了友善的逐帧编辑的页面。 下载地址：http://hoxis-github-io.qiniudn.com/ScreenToGif.exe

    licecap¶

    一款屏幕录制工具，支持导出GIF动画图片格式，轻量级、使用简单，录制过程中可以随意改变录屏范围。 下载地址：http://hoxis-github-io.qiniudn.com/licecap128-install.exe

    PanDownload¶

    百度网盘全速下载神器 下载地址：http://pandownload.com/

    hfs¶

    局域网文件传输，网络文件服务器 PC 版是专门为个人用户所设计的 Http File Server HTTP 档案系统工具，hfs 网络文件服务器 PC 版没有建立像ftp服务器那样复杂，也不需要安装，每次使用只要双击打开就行了，设置好IP和目录，操作上也很方便，设置的功能超多。 下载地址：http://hoxis-github-io.qiniudn.com/hfs.exe
