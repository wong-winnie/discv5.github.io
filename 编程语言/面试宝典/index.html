  • A. p->link = q->link; q = p->link;
  • B. p->link = q; q->link = p->link;
  • C. q->link = p->link; p->link = q;
  • D. q = p->link; p->link = q->link;

( C ) 首先先把q->link=p->link; 然后旧节点的link指向新节点 p->link=q

二、区块链主链技术考察（请用自己的语言来描述，不要用百度结果）：

1、解释一下 UTXO（请用自己语言描述，下同）。

未消费的交易输出: 遍历个人账户地址的交易记录得出最后的计算结果

2、解释一下简单支付验证（SPV）过程。

比特币支付验证： 下载区块头部找到要验证的这笔交易在哪个区块； 确定这个交易是否被6次确认过了。 以太坊为12次安全验证。

3、为何以太坊不使用 SPV？

跟账户模型有关，spv需要手续费来支持，防止欺诈和安全问题

4、解释一下ECDH密钥交换协议（算法）。

ECC算法和DH结合使用，用于密钥磋商，这个密钥交换算法称为ECDH。交换双方可以在不共享任何秘密的情况下协商出一个密钥。ECC是建立在基于椭圆曲线的离散对数问题上的密码体制，给定椭圆曲线上的一个点P，一个整数k，求解Q=kP很容易；给定一个点P、Q，知道Q=kP，求整数k确是一个难题。
